STM32H735G-DK board

The CANopen FD Master protocol stack for the STM32H735G-DK runs on a bit-rate of 500 kBit/s on interface CAN1.

Prerequisites

The example project can be built if the following programs are installed:

  • GNU Arm Embedded Toolchain
  • CMake

Please make sure that the value of WIN_TOOLCHAIN_PATH (for Windows) or UNIX_TOOLCHAIN_PATH (for macOS / Linux) inside the cmake/toolchain-gcc-<version>.cmake file is equal to the install path of the GNU Arm Embedded Toolchain.

How to build

Create a local build directory and start the build process within this directory. Make sure to pass the correct toolchain (i.e. GNU Arm Embedded Toolchain) to the CMake process via the parameter -DCMAKE_TOOLCHAIN_FILE. The following examples assume a GNU Arm Embedded Toolchain version 9.2.1.

Debug build

mkdir build
cd build
cmake -DCMAKE_TOOLCHAIN_FILE=../../../../../../cmake/toolchain-gcc-10.2.1.cmake -DCMAKE_BUILD_TYPE=Debug ..
cmake --build .

Release build

mkdir build
cd build
cmake -DCMAKE_TOOLCHAIN_FILE=../../../../../../cmake/toolchain-gcc-10.2.1.cmake ..
cmake --build .

STM32H735G-DK board settings

The CANopen FD demo uses CAN1 for communication, i.e. the connection is done via connector CN18 on the evaluation board. The LEDs LD1 (green) and LD3 (red) are used to display the CANopen NMT state and CAN error status. The default bitrate in the demo application is 500 kBit/s, the demo is stating up in CANopen FD mode.

Firmware flashing and debugging

The firmware can be flashed via the on-board ST-Link/V3 connector (CN15). When using openOCD the configuration file oocd_stlink.cfg is located in the demo directory.

For debugging the firmware must be compiled as debug build. Users of VS Code may use the following launch configuration for debugging:

{
"name": "Debug STM32H735G-DK",
"type": "cortex-debug",
"request": "launch",
"servertype": "openocd",
"cwd": "${workspaceRoot}",
"executable": "${workspaceRoot}/source/application/examples-com/boards/STM32H735G-DK/build/com-stm32h735g.elf",
"device": "STM32H735G",
"configFiles": [
"${workspaceRoot}/source/application/examples-com/boards/STM32H735G-DK/oocd_stlink.cfg"
],
"runToMain": true
}
