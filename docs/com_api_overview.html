CANopen (FD) Master API overview

All header files and source files of the CANopen (FD) Master protocol stack package are located inside the source directory. The following picture shows an overview of the source code structure.

Directory structure

The files inside the source/application directory shall be used to adopt the CANopen (FD) Master protocol stack to the target. All other files shall not be modified by the user.

Requirements

The CANopen (FD) Master required source files, are in the stack-com, misc and canpie-fd directory.
