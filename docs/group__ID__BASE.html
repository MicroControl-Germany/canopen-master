Detailed Description

The ID_BASE definitions specify the starting CAN-ID value (offset) for various CANopen (FD) services (EMCY, SDO, etc).

Macro Definition Documentation

◆ ID_BASE_EMCY

#define ID_BASE_EMCY   0x080

Emergency (EMCY) Identifier.

The Emergency identifier for a CANopen device is calculated by the formula node-ID + ID_BASE_EMCY.

◆ ID_BASE_NMT_ERR

#define ID_BASE_NMT_ERR   0x700

NMT error control Identifier.

The NMT error control identifier (heartbeat) for a CANopen device is calculated by the formula node-ID + ID_BASE_NMT_ERR.
