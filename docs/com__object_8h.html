enum ComObjectMarker_e

SDO / USDO transfer marker.

The enumeration defines values for the marker field inside the CANopen object structure CoObject_s. The marker field can be evaluated inside the ComSdoEventObjectReady() or ComUsdoEventObjectReady() function.
