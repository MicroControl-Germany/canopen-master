Detailed Description

This file implements an local object dictionary for the CANopen (FD) Master protocol stack. The following table shows all objects supported by this example.

The table is added to the CANopen (FD) Master stack by means of the ComDictAddList() function.

//--------------------------------------------------------------------------
// Initialise the CANopen Master stack with address 1, bit-rate 500 kBit/s
//
ComMgrInit(eCP_CHANNEL_1, eCOM_NET_1, eCP_BITRATE_500K, 1, eCOM_MODE_NMT_MASTER);
//--------------------------------------------------------------------------
// add local object dictionary
//
ComDictAddList(eCOM_NET_1, &atsComDictMasterTableG[0]);
